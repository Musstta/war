# Scenario: tribute-diminishing

> Nicaragua (poor, wealthStock=5) pays 10 Wealth/tick tribute to Honduras (rich, wealthStock=100). With TRIBUTE_DIMINISHING_SCALE=100 and TRIBUTE_DIMINISHING_CAP=0.5, Honduras's diminishFactor = min(0.5, 100/100) = 0.5, so effectiveTribute = 10 × (1−0.5) = 5. Nicaragua loses 10/tick but Honduras gains only 5/tick. Costa Rica (single territory, wealthStock=0) pays 10/tick tribute to Guatemala (wealthStock=0) — no diminishing at all (diminishFactor=0), so Guatemala gains full 10/tick. Assert: after 5 ticks, Honduras wealthStock ≈ 100 + 5×5 = 125 (vs 150 without diminishing); Guatemala wealthStock ≈ 0 + 5×10 = 50. Also: payer/receiver territory-count ratio triggers exploitation check — Nicaragua (1 territory) vs Honduras (1 territory) = ratio 1.0 ≥ threshold 0.4 → NOT exploitative. Guatemala (1 terr) vs Costa Rica (1 terr) = 1.0 → NOT exploitative.
